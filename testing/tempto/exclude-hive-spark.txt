# Hive-metastore / Spark / HDFS coupled test classes to exclude from the
# "iceberg REST catalog" full run (scripts/tempto-test.sh --rest).
#
# SQE is a pure Iceberg REST-catalog engine: no Hive metastore, no Spark, no
# HDFS. The upstream io.trino...iceberg product-tests multi-tag their classes
# (a class can carry iceberg_rest AND iceberg_jdbc AND iceberg_nessie), so group
# selection alone cannot separate the REST-catalog tests from the Hive/Spark
# ones. These classes inject a ThriftMetastoreClient or need @Named
# databases.hive.* / a Spark session in setUp/tearDown, so every one of their
# methods fails at fixture time (NullPointerException / Guice
# ConfigurationException) before any SQL reaches SQE. Excluding them removes the
# ~85% environmental noise and leaves the tests that actually exercise SQE's
# REST catalog.
#
# Format: one dot-separated test-name prefix per line (passed to tempto
# --excluded-tests). Blank lines and #-comments are ignored.
#
# Verified: with these excluded, `--groups iceberg` produces zero
# `metastoreClient is null` and zero `databases.hive.metastore.host` failures.
io.trino.tests.product.iceberg.TestIcebergSparkCompatibility
io.trino.tests.product.iceberg.TestIcebergSparkDropTableCompatibility
io.trino.tests.product.iceberg.TestIcebergRedirectionToHive
io.trino.tests.product.iceberg.TestIcebergHiveViewsCompatibility
io.trino.tests.product.iceberg.TestIcebergHiveMetadataListing
io.trino.tests.product.iceberg.TestIcebergFormatVersionCompatibility
