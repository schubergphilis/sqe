# Cover Art Prompts — Sovereign by Design

Use these prompts with Midjourney, DALL-E, Ideogram, or similar tools. Each variant captures a different aspect of the book.

---

## Prompt 1: The Architecture (technical, clean)

```
Book cover for a technical book called "Sovereign by Design." A minimalist architectural blueprint showing a distributed query engine — a central coordinator node connected to worker nodes by glowing data streams, all sitting on top of layered Iceberg-like crystal structures representing data tables. Dark navy background with teal and amber accent lines. No text. Clean geometric style, reminiscent of O'Reilly technical book covers but more modern. Subtle Rust orange accent color. A5 portrait format.
```

## Prompt 2: The Explorer (adventure, problem-solver)

```
Book cover art. A lone engineer standing at the edge of a vast data landscape — geometric crystalline structures representing Iceberg tables stretching to the horizon, with glowing pathways connecting them like neural networks. The engineer holds a compass (representing sovereignty — choosing your own direction). Dramatic lighting from below, dark sky with constellation-like node patterns. Muted color palette: deep navy, teal, warm amber. No text. Cinematic composition, A5 portrait. Style: detailed digital illustration, not photorealistic.
```

## Prompt 3: The Key (sovereignty metaphor)

```
Technical book cover. A single ornate key made of interlocking Rust-colored gears and circuit patterns, floating above a reflection pool that shows a distributed network topology. The key represents sovereignty — owning your own infrastructure. Background: deep dark blue gradient. The key glows with warm amber light. Minimalist, elegant, slightly Art Deco influence. No text. A5 portrait format.
```

## Prompt 4: The Split (distributed execution)

```
Abstract book cover art. A single luminous query plan tree — nodes and edges made of light — being split into fragments that flow to three separate execution points. The fragments carry different colors representing different workers. The central node (coordinator) is larger and brighter. Dark background with a subtle grid pattern. Colors: electric teal for the plan, warm amber for the coordinator, cool blue for workers. Geometric, precise, technical aesthetic. No text. A5 portrait.
```

## Prompt 5: The Token (authentication/identity)

```
Book cover. A bearer token visualised as a golden thread that passes through a series of transparent architectural layers — a catalog layer, a query engine layer, a storage layer. Each layer is a different translucent geometric shape. The thread connects them all, unbroken. The layers float in dark space. The golden thread is the only warm color in an otherwise cool blue-grey palette. Minimalist, premium feel. No text. A5 portrait format.
```

## Prompt 6: The Rust Forge (building metaphor)

```
Technical book cover. A blacksmith's anvil made of circuit board patterns, with a glowing query plan being forged on it. Sparks of Arrow-shaped data particles flying off. The anvil sits in a dark workshop with subtle DataFusion/Rust iconography in the background. Warm orange forge light contrasts with cool blue data streams. Industrial meets digital. Dramatic chiaroscuro lighting. No text. A5 portrait.
```

## Prompt 7: The Iceberg (literal, elegant)

```
Book cover. A crystalline iceberg seen from the side — the small visible tip above the waterline represents the SQL query, while the massive underwater structure shows the metadata tree: manifest lists, manifests, and thousands of data files arranged in geometric precision. The water surface is a clean horizontal line dividing the composition. Above: warm light, simple. Below: vast, structured, blue-green. Ultra-clean composition. No text. A5 portrait.
```

## Style Notes for All Prompts

- **Format:** A5 portrait (148mm × 210mm), 300 DPI for print
- **Color palette:** Navy (#1a237e), Teal (#00695c), Amber (#ff8f00), Rust orange (#d84315), White
- **Avoid:** Stock photo aesthetics, generic "tech" imagery (circuit boards, binary code rain), AI slop (too much detail, melting edges)
- **Target feel:** Premium, technical, intelligent. Think O'Reilly animal covers meets Dieter Rams design.
- **The title "Sovereign by Design" will be overlaid in Source Serif Pro, so leave clean space at the top third of the composition**
